# ℹ Description
This Open Source Project was inspired on Self-Organizing Maps (Kohonen's Maps). It's application is to clustering all of the properties of concrete that a civil engineer want to classificate and identifying those patterns, focused to tell if that concrete is availabe to manipulate or not.

# 🤖 This project was developed using those technologies:

- Python
- MiniSom library available in: https://github.com/JustGlowing/minisom/
